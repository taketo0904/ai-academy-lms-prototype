# AI ACADEMY — eラーニング プロトタイプ

自社AIスクール向けのeラーニングサービス プロトタイプ。ブラウザでログインしてコースを回遊でき、**動画コースに加えてAIエージェント/スキルをその場で受け取り・譲渡できる**仕組みを備える。UIはSHIFT AIを参考にしたイメージ。

## 使い方

`index.html` をブラウザで開くだけ（ビルド不要・依存なしの単一HTML）。

- ログイン画面のメール/パスワードは入力済み。そのまま「ログイン」で入れる（プロトタイプのためモック認証）。
- 受け取り済みエージェントと学習進捗はブラウザの `localStorage` に保存される。

ローカルで開くには：

```bash
python3 -m http.server 4599
# → http://localhost:4599/
```

## 機能

- **ログイン**（モック認証）
- **コースライブラリ** — カテゴリ絞り込み / グリッド⇄リスト切替 / 検索 / 進捗バー
- **コース詳細** — 動画プレイヤー枠 + セクションのアコーディオン + レッスン完了トグル
- **スキルストア** — AIエージェントを「その場で受け取る（譲渡）」→ マイエージェントへ
- **譲渡フロー** — 受け取ったエージェントを他の受講者へ譲渡（所有が移動）

## カスタマイズ（差し替えポイント）

`index.html` 内の以下の配列を書き換えるだけで内容が更新される。

- `COURSES` … コース・セクション・レッスン
- `AGENTS` … 配布するエージェント/スキル（`course` でコースに紐づけ）
- `CATEGORIES` … 左サイドのカテゴリ

## 未確定の設計論点

「譲渡」の意味をどちらにするかで課金・ライセンス設計が変わる。

- **A案（現状の実装）**: 譲渡＝所有の移動。渡すと自分の手元から消える
- **B案**: 配布＝複製を渡す。自分も相手も持てる（講師→受講者へ一斉付与）
